sedflux Modules
===============

sedflux comes with a number of process modules.  Apart from running within the
sedflux framework, many of these process modules can be run as standalone
programs as well.  This page is intended to provide links to descriptions of
these modules (as both sedflux modules and standalone programs).

 * :ref:`Plume <sedflux_module_plume>`: Hypopycnal (surface) sediment plumes
 * :ref:`Avulsion <sedflux_module_avulsion>`: River avulsion on a delta
 * :ref:`Subsidence <sedflux_module_subsidence>`: Specify subsidence rates in time and space
 * :ref:`Isostasy <sedflux_module_isostasy>`: Flexure due to sediment and water loading
 * :ref:`Bioturbation <sedflux_module_bioturbation>`: Reworking of the seafloor by worms and the like
 * :ref:`Sea Level <sedflux_module_sea_level>`: Specify base level change
 * :ref:`Bed load dumping <sedflux_module_bed_load>`: Deposit a river's bed load
 * :ref:`Erosion <sedflux_module_erosion>`: Erode subaerial sediments of a delta
 * :ref:`Compaction <sedflux_module_compaction>`: Compact sediment due to overlying load
 * :ref:`Data Dump <sedflux_module_data_dump>`: Writes a sedflux data cube to a file
 * :ref:`Measuring Station <sedflux_module_measuring_station>`: Writes sedflux sea-floor data to a file
 * :ref:`Sediment Failure <sedflux_module_failure>`: Look for slope failures
 * :ref:`Debris Flows <sedflux_module_debris_flow>`: Move failed sediment as a debris flow
 * :ref:`Turbidity Currents <sedflux_module_turbidity_current>`: Move failed sediment as a turbidity current
 * :ref:`Ocean storms <sedflux_module_ocean_storms>`: Generate stormy waves
 * :ref:`Cross-Shore Transport <sedflux_module_xshore>`: Move sediment with waves/currents
